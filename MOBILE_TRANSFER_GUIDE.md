## 1. ภาพรวมระบบ

### ระบบ Mobile Transfer คืออะไร?
ระบบสำหรับพนักงานคลังใช้งานบนมือถือหรือแท็บเล็ต เพื่อ **ดำเนินการย้ายสินค้า** ตามใบย้ายที่ได้รับมอบหมาย

### ฟีเจอร์หลัก
✅ แสดงรายการใบย้ายทั้งหมด
✅ กรองและค้นหาใบย้าย
✅ ดูรายละเอียดใบย้าย
✅ สแกนพาเลท
✅ ยืนยันจำนวนสินค้า
✅ สแกนโลเคชั่นปลายทาง
✅ แสดงความคืบหน้า
✅ เสียงยืนยันการกดปุ่ม

### ใครใช้งาน?
- พนักงานดำเนินการ (Operator)
- พนักงานหยิบสินค้า (Picker)
- พนักงานขับโฟล์คลิฟท์ (Forklift)
- หัวหน้าคลัง (Supervisor)

## 2. การเข้าใช้งาน

### 2.1 เปิดแอพ

1. เปิดเว็บเบราว์เซอร์บนมือถือ (Chrome, Safari)
2. ไปที่ URL: `http://localhost:3000/mobile/transfer`
3. Login ด้วยบัญชีพนักงาน

### 2.2 เพิ่ม Shortcut ลงหน้าจอหลัก (iOS)

**iPhone/iPad:**
1. เปิด Safari
2. กด "Share" (ไอคอนแชร์)
3. เลือก "Add to Home Screen"
4. ตั้งชื่อ "Transfer"
5. กด "Add"

**Android:**
1. เปิด Chrome
2. กด "Menu" (3 จุด)
3. เลือก "Add to Home screen"
4. ตั้งชื่อ "Transfer"
5. กด "Add"

### 3.2 สถิติด้านบน

**4 การ์ดสถิติ:**
- **ทั้งหมด**: จำนวนใบย้ายทั้งหมด
- **มอบหมาย**: ใบย้ายที่มอบหมายแล้ว (สีฟ้า)
- **ดำเนินการ**: กำลังดำเนินการ (สีเหลือง)
- **เสร็จสิ้น**: ทำเสร็จแล้ว (สีเขียว)

### 3.3 การค้นหา

**วิธีค้นหา:**
1. กดช่องค้นหา
2. พิมพ์เลขใบย้าย เช่น `MV-2025-0001`
3. ระบบจะกรองผลลัพธ์ทันที

**ค้นหาได้:**
- เลขใบย้าย (move_no)
- เลขอ้างอิง (reference_doc_no)

### 3.4 ตัวกรอง

**กดปุ่ม "ตัวกรอง" เพื่อเปิด/ปิดตัวเลือก**

**กรองตามสถานะ:**
- ทุกสถานะ
- รอดำเนินการ
- มอบหมายแล้ว
- กำลังดำเนินการ
- เสร็จสิ้น
- ยกเลิก

**กรองตามประเภท:**
- ทุกประเภท
- จัดเก็บสินค้า
- ย้ายสินค้า
- เติมสินค้า
- ย้ายตำแหน่ง

**ล้างตัวกรอง:**
กดปุ่ม "ล้างตัวกรอง" เพื่อรีเซ็ต

### 3.5 การ์ดใบย้าย

**ข้อมูลในการ์ด:**
- เลขใบย้าย (เช่น MV-2025-0001)
- Badge สถานะ (สีต่างๆ)

**การกดเปิด:**

## 5. ขั้นตอนการย้ายสินค้า

### Step 1: สแกนพาเลท (Scan Pallet)

**วิธีทำ:**
1. เปิดใบย้ายในหน้า Detail
2. ระบบจะอยู่ในขั้นตอน "สแกน Pallet"
3. สแกนบาร์โค้ดบนพาเลท หรือ
4. พิมพ์รหัสพาเลทด้วยตนเอง
5. กดปุ่ม "สแกน"

**ผลลัพธ์:**
- ✅ ถูกต้อง → ไปขั้นตอนถัดไป
- ❌ ผิด → แสดงข้อความแจ้งเตือน

**หมายเหตุ:**
- ต้องสแกนให้ตรงกับพาเลทที่ระบุในใบย้าย
- ถ้าพาเลทไม่ตรง จะไม่สามารถดำเนินการต่อได้

### Step 2: ยืนยันจำนวน (Confirm Quantity)

**วิธีทำ:**
1. ตรวจสอบจำนวนสินค้าบนพาเลท
2. ปรับจำนวนถ้าไม่ตรง (กด +/-)
3. กดปุ่ม "ยืนยันจำนวน"

**ข้อมูลที่แสดง:**
- จำนวนที่ร้องขอ (เป้าหมาย)
- จำนวนที่ยืนยัน (ปรับได้)

**กรณีจำนวนไม่ตรง:**
- ระบบจะถามยืนยันก่อนดำเนินการต่อ
- สามารถเลือก "ดำเนินการต่อ" หรือ "ยกเลิก"

### Step 3: สแกนโลเคชั่นปลายทาง (Scan To Location)

**วิธีทำ:**
1. นำพาเลทไปวางที่ตำแหน่งปลายทาง
2. สแกน QR Code ที่ตำแหน่งปลายทาง หรือ
3. พิมพ์รหัสโลเคชั่นด้วยตนเอง
4. กดปุ่ม "สแกน"

**ผลลัพธ์:**
- ✅ ถูกต้อง → เสร็จสิ้นรายการนี้
- ❌ ผิด → แสดงข้อความแจ้งเตือน

**หลังสแกนสำเร็จ:**
- แสดงข้อความ "ย้ายสินค้าเรียบร้อย" 1.5 วินาที
- กลับไปขั้นตอนที่ 1 เพื่อทำรายการถัดไป
- อัพเดท Progress Bar

### Step 4: เสร็จสิ้นทั้งใบย้าย

**เมื่อทำครบทุกรายการ:**
1. Progress Bar จะแสดง 100%
2. ปุ่ม "เสร็จสิ้นการย้ายสินค้า" จะปรากฏ
3. กดปุ่มเพื่อยืนยัน
4. กลับไปหน้ารายการ (List)

## 7. คำถามที่พบบ่อย (FAQ)

### Q1: ทำไมไม่เห็นใบย้ายของตัวเอง?

**A:** ตรวจสอบดังนี้
1. เช็คว่า Login ด้วยบัญชีที่ถูกต้อง
2. เช็คตัวกรองสถานะ (ลองเลือก "ทุกสถานะ")
3. กดปุ่ม "รีเฟรชข้อมูล"
4. ติดต่อหัวหน้าคลังถ้ายังไม่เห็น

### Q2: สแกนพาเลทแล้วบอกว่าไม่ตรง?

**A:** เช็คดังนี้
1. พาเลทที่สแกนตรงกับที่ระบุในใบย้ายหรือไม่
2. พาเลทอาจอยู่ในใบย้ายอื่น
3. ลองพิมพ์รหัสพาเลทด้วยตนเอง
4. ติดต่อหัวหน้าคลังถ้ายังมีปัญหา

### Q3: จำนวนสินค้าไม่ตรงกับในใบย้าย?

**A:** ทำดังนี้
1. นับสินค้าบนพาเลทอีกครั้ง
2. ปรับจำนวนในหน้าจอให้ตรงกับจำนวนจริง
3. กดปุ่ม "ยืนยันจำนวน"
4. ระบบจะถามยืนยัน → เลือก "ดำเนินการต่อ"
5. บันทึกหมายเหตุว่าทำไมจำนวนไม่ตรง

### Q4: สแกนโลเคชั่นปลายทางแล้วบอกว่าไม่ตรง?

**A:** เช็คดังนี้
1. ตรวจสอบว่าวางพาเลทที่ตำแหน่งที่ถูกต้อง
2. สแกน QR Code ที่ตำแหน่งให้ถูกต้อง
3. ลองพิมพ์รหัสโลเคชั่นด้วยตนเอง
4. ถ้ายังไม่ได้ → ติดต่อหัวหน้าคลัง

### Q5: ต้องการเปลี่ยนพาเลทที่กำลังทำอยู่?

**A:** ทำไม่ได้ครับ
- ต้องทำพาเลทปัจจุบันให้เสร็จก่อน
- หรือยกเลิกใบย้ายทั้งหมด (ติดต่อหัวหน้าคลัง)

### Q6: Progress Bar ไม่เปลี่ยน?

**A:** ทำดังนี้
1. กดปุ่ม "รีเฟรชข้อมูล"
2. ออกจากหน้า Detail แล้วเข้าใหม่
3. ถ้ายังไม่เปลี่ยน → ติดต่อ IT

### Q7: เสียงกดปุ่มไม่มี?

**A:** ตรวจสอบดังนี้
1. เปิดเสียงมือถือ
2. ปรับระดับเสียงให้พอเหมาะ
3. เบราว์เซอร์บางตัวอาจบล็อกเสียงอัตโนมัติ

### Q8: หน้าจอโหลดช้า?

**A:** ทำดังนี้
1. เช็คสัญญาณ Wi-Fi / 4G
2. ปิดแท็บอื่นๆ ในเบราว์เซอร์
3. รีสตาร์ทเบราว์เซอร์
4. ติดต่อ IT ถ้ายังช้า

## 8. การแก้ปัญหาเบื้องต้น

### 8.1 หน้าจอขาว/ไม่โหลด

**วิธีแก้:**
1. รีเฟรชหน้า (Pull to refresh)
2. ปิดแอพและเปิดใหม่
3. Clear cache ของเบราว์เซอร์
4. รีสตาร์ทมือถือ

### 8.2 ไม่สามารถ Login

**วิธีแก้:**
1. เช็ค Username/Password
2. เช็คสัญญาณอินเทอร์เน็ต
3. ติดต่อ IT เพื่อ reset password

### 8.3 กล้องสแกนไม่ทำงาน

**วิธีแก้:**
1. ตรวจสอบการอนุญาตกล้อง (Camera permission)
2. ลองพิมพ์รหัสด้วยตนเองแทนการสแกน
3. ใช้แอพสแกนภายนอกแล้วคัดลอกรหัสมา

### 8.4 ข้อมูลไม่ update

**วิธีแก้:**
1. กดปุ่ม "รีเฟรชข้อมูล"
2. ออกจากหน้าแล้วเข้าใหม่
3. ติดต่อ IT

### 8.5 Error Message

**ข้อความที่พบบ่อย:**

| Error | ความหมาย | วิธีแก้ |
|-------|----------|---------|
| "ไม่พบข้อมูลรายการย้ายสินค้า" | ใบย้ายไม่มีในระบบ | เช็คเลขใบย้าย |
| "ไม่พบ Pallet ID นี้" | พาเลทไม่อยู่ในใบนี้ | เช็คพาเลท |
| "ตำแหน่งปลายทางไม่ตรงกัน" | สแกนผิดโลเคชั่น | สแกนใหม่ |
| "Failed to fetch" | ไม่มีอินเทอร์เน็ต | เช็คสัญญาณ |
