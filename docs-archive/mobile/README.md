# 📱 Mobile Documentation

เอกสารคู่มือระบบมือถือสำหรับคลังสินค้า

## เอกสารในโฟลเดอร์นี้

### คู่มือการใช้งาน
- **MOBILE_RECEIVE_GUIDE.md** - คู่มือระบบรับสินค้าบนมือถือแบบละเอียด
- **MOBILE_TRANSFER_GUIDE.md** - คู่มือระบบย้ายสินค้าระหว่างสถานที่บนมือถือ

### การวิเคราะห์ระบบ
- **INBOUND_ANALYSIS_FOR_MOBILE.md** - การวิเคราะห์ระบบรับสินค้าเพื่อพัฒนา mobile interface
- **TRANSFER_ANALYSIS_FOR_MOBILE.md** - การวิเคราะห์ระบบย้ายสินค้าเพื่อพัฒนา mobile interface

## ใช้เมื่อไหร่?

### สำหรับ Developer Mobile
ใช้เอกสารเหล่านี้เมื่อ:
- พัฒนาหน้า Mobile ใหม่
- แก้ไข UI/UX สำหรับ mobile
- ทำความเข้าใจ workflow การทำงานบนมือถือ

### สำหรับ UX/UI Designer
ใช้เอกสารเหล่านี้เมื่อ:
- ออกแบบ interface สำหรับงานคลัง
- เข้าใจ user flow ของพนักงานคลัง
- ปรับปรุง user experience

## เอกสารที่เกี่ยวข้อง

- `DESIGN_SYSTEM.md` (root) - ระบบดีไซน์ UI/UX
- `components/mobile/*` - Mobile components
- `app/mobile/*` - Mobile pages
