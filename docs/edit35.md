## 🔍 PHASE 1: ตรวจสอบระดับ PALLET (ทุกพาเลท - ห้ามข้าม)

### 1.1 สำหรับทุก pallet_id ใน UNIQUE_PALLETS:

```
FOR EACH pallet_id IN UNIQUE_PALLETS:
    
    1.1.1 ดึง Ledger ทั้งหมดของพาเลทนี้
    ┌─────────────────────────────────────────────────────────────┐
    │ SELECT * FROM ประวัติเคลื่อนไหว                             │
    │ WHERE รหัสพาเลท = '{pallet_id}'                             │
    │ ORDER BY วันที่/เวลา ASC                                     │
    └─────────────────────────────────────────────────────────────┘
    
    1.1.2 คำนวณ Running Balance
    ┌─────────────────────────────────────────────────────────────┐
    │ running_balance = 0                                          │
    │ FOR EACH entry IN ledger:                                    │
    │     IF ทิศทาง = 'เข้า':                                      │
    │         running_balance += ชิ้น                              │
    │     ELIF ทิศทาง = 'ออก':                                     │
    │         running_balance -= ชิ้น                              │
    │     PRINT(f"{entry.ID} | {entry.ประเภท} | {entry.ทิศทาง} |  │
    │            {entry.ชิ้น} | running={running_balance}")        │
    └─────────────────────────────────────────────────────────────┘
    
    1.1.3 ดึง Balance ปัจจุบันของพาเลทนี้
    ┌─────────────────────────────────────────────────────────────┐
    │ SELECT * FROM ยอดสต็อกคงเหลือ                               │
    │ WHERE รหัสพาเลท = '{pallet_id}'                             │
    └─────────────────────────────────────────────────────────────┘
    
    1.1.4 เปรียบเทียบ
    ┌─────────────────────────────────────────────────────────────┐
    │ IF running_balance != balance.ชิ้นรวม:                       │
    │     ISSUE: "พาเลท {pallet_id} Ledger Sum={running_balance}  │
    │            แต่ Balance={balance.ชิ้นรวม}"                    │
    │     DIFF = balance.ชิ้นรวม - running_balance                 │
    │     ROOT_CAUSE = ???                                         │
    └─────────────────────────────────────────────────────────────┘
    
    1.1.5 ตรวจว่าพาเลทอยู่กี่ Location
    ┌─────────────────────────────────────────────────────────────┐
    │ SELECT ตำแหน่ง, COUNT(*), SUM(ชิ้นรวม)                       │
    │ FROM ยอดสต็อกคงเหลือ                                        │
    │ WHERE รหัสพาเลท = '{pallet_id}'                             │
    │ GROUP BY ตำแหน่ง                                            │
    │                                                              │
    │ IF COUNT(DISTINCT ตำแหน่ง) > 1:                              │
    │     ISSUE: "พาเลท {pallet_id} อยู่หลาย Location"            │
    └─────────────────────────────────────────────────────────────┘

END FOR
```

## 🔍 PHASE 2: ตรวจสอบ TRANSFER PAIRS (ทุก Move ID)

### 2.1 สำหรับทุก move_id ใน UNIQUE_MOVE_IDS:

```
FOR EACH move_id IN UNIQUE_MOVE_IDS:
    
    2.1.1 ดึง Ledger entries ของ Move นี้
    ┌─────────────────────────────────────────────────────────────┐
    │ SELECT * FROM ประวัติเคลื่อนไหว                             │
    │ WHERE Move_ID = '{move_id}'                                  │
    │ ORDER BY ทิศทาง                                              │
    └─────────────────────────────────────────────────────────────┘
    
    2.1.2 แยก เข้า vs ออก
    ┌─────────────────────────────────────────────────────────────┐
    │ entries_in = [e for e in entries if e.ทิศทาง == 'เข้า']     │
    │ entries_out = [e for e in entries if e.ทิศทาง == 'ออก']     │
    │                                                              │
    │ sum_in = sum(e.ชิ้น for e in entries_in)                     │
    │ sum_out = sum(e.ชิ้น for e in entries_out)                   │
    └─────────────────────────────────────────────────────────────┘
    
    2.1.3 ตรวจสอบจับคู่
    ┌─────────────────────────────────────────────────────────────┐
    │ IF len(entries_in) == 0:                                     │
    │     ISSUE: "Move {move_id} ไม่มี entry เข้า"                │
    │ IF len(entries_out) == 0:                                    │
    │     ISSUE: "Move {move_id} ไม่มี entry ออก"                 │
    │ IF sum_in != sum_out:                                        │
    │     ISSUE: "Move {move_id} เข้า={sum_in} ออก={sum_out}"    │
    │     DIFF = sum_in - sum_out                                  │
    └─────────────────────────────────────────────────────────────┘
    
    2.1.4 ตรวจสอบ Pallet/SKU/Location
    ┌─────────────────────────────────────────────────────────────┐
    │ FOR entry_in IN entries_in:                                  │
    │     # หา entry_out ที่ match                                 │
    │     matching_out = find(entries_out,                         │
    │         pallet = entry_in.รหัสพาเลท,                        │
    │         sku = entry_in.รหัสสินค้า,                          │
    │         qty = entry_in.ชิ้น)                                 │
    │     IF NOT matching_out:                                     │
    │         ISSUE: "entry เข้า ID={entry_in.ID} ไม่มีคู่ออก"   │
    └─────────────────────────────────────────────────────────────┘

END FOR
```

## 🔍 PHASE 4: ตรวจสอบ ORDER → PICK → LOAD (ทุกออเดอร์)

### 4.1 สำหรับทุก order_number:

```
FOR EACH order IN ชีท_order:
    
    4.1.1 รวบรวมสินค้าในออเดอร์
    ┌─────────────────────────────────────────────────────────────┐
    │ order_items = {}                                             │
    │ FOR row IN order_rows WHERE เลขออเดอร์ == order.เลขออเดอร์: │

    ┌─────────────────────────────────────────────────────────────┐

    │     picked_qty = sum(ledger.ชิ้น WHERE sku = sku)            │
    │     IF ordered_qty != picked_qty:                            │
    │         ISSUE: "Order {order} SKU {sku}                      │
    │                สั่ง={ordered_qty} หยิบ={picked_qty}"        │
    └─────────────────────────────────────────────────────────────┘
    
    4.1.5 หา Loadlist ที่เกี่ยวข้อง
    ┌─────────────────────────────────────────────────────────────┐
    │ SELECT DISTINCT Loadlist                                     │
    │ FROM โหลดสินค้าเสร็จ                                        │
    │ WHERE เลขออเดอร์ = '{order.เลขออเดอร์}'                     │
    └─────────────────────────────────────────────────────────────┘
    
    4.1.6 เปรียบเทียบ Picked vs Loaded
    ┌─────────────────────────────────────────────────────────────┐
    │ FOR sku IN order_items:                                      │

    │     loaded = โหลดสินค้าเสร็จ[sku]                            │

    │         ISSUE: "Order {order} SKU {sku}                      │
```

```
│     IF ledger_count == 0:                                    │
```

## 🔍 PHASE 6: ตรวจสอบความสัมพันธ์ข้ามชีท (Cross-Sheet)

### 6.1 บ้านหยิบ vs ยอดสต็อกคงเหลือ

```
FOR EACH row IN บ้านหยิบ:
    
    # หาใน ยอดสต็อกคงเหลือ
    balance = SELECT * FROM ยอดสต็อกคงเหลือ
              WHERE รหัสพาเลท = row.รหัสพาเลท
              AND รหัสสินค้า = row.รหัสสินค้า
              AND ตำแหน่ง = row.ตำแหน่ง
    
    IF NOT balance:
        ISSUE: "บ้านหยิบ Balance_ID={row.ID} ไม่มีใน ยอดสต็อกคงเหลือ"
    ELIF balance.ชิ้นรวม != row.จำนวนชิ้น:
        ISSUE: "บ้านหยิบ vs Balance ไม่ตรง:
               บ้านหยิบ={row.จำนวนชิ้น} Balance={balance.ชิ้นรวม}"
END FOR
```

### 6.2 จัดสินค้าเสร็จ vs Ledger

```
FOR EACH row IN จัดสินค้าเสร็จ:
    
    # หา Ledger entry ที่ตรงกัน
    ledger = SELECT * FROM ประวัติเคลื่อนไหว
             WHERE รหัสสินค้า = row.รหัสสินค้า
             AND ตำแหน่ง = 'Dispatch'
             AND ทิศทาง = 'เข้า'
    
    # ตรวจสอบยอดสะสม
    dispatch_sum = SUM(ledger.ชิ้น WHERE ทิศทาง='เข้า') - 
                   SUM(ledger.ชิ้น WHERE ทิศทาง='ออก')
    
    IF dispatch_sum != row.จำนวนชิ้น:
        ISSUE: "Dispatch SKU={row.รหัสสินค้า}
               Ledger sum={dispatch_sum} Dispatch={row.จำนวนชิ้น}"
END FOR
```

### 6.3 โหลดสินค้าเสร็จ vs Order

```
FOR EACH order IN UNIQUE_ORDERS:
    
    # รวมจาก order sheet
    ordered = SUM(order.จำนวน)
    
    # รวมจาก โหลดสินค้าเสร็จ
    loaded = SUM(โหลดสินค้าเสร็จ.จำนวนหยิบ 
                 WHERE เลขออเดอร์ = order)
    
    IF ordered != loaded:
        ISSUE: "Order {order} สั่ง={ordered} โหลด={loaded}"
END FOR
```
