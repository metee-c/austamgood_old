## ภาพรวมระบบ

ระบบจัดการแผนเส้นทาง (Route Planning) เป็นเครื่องมือที่ช่วยให้คุณสามารถ:

- **สร้างแผนเส้นทาง** - วางแผนการจัดส่งสินค้าอัตโนมัติด้วย AI
- **จัดการเส้นทาง** - แก้ไข ย้าย หรือแบ่งจุดส่งได้ง่าย
- **ติดตามสถานะ** - ดูสถานะแผนแบบ real-time
- **วิเคราะห์ข้อมูล** - ดูรายงานระยะทาง ต้นทุน และประสิทธิภาพ
- **ส่งออกข้อมูล** - Export เป็น Excel สำหรับพิมพ์หรือแชร์

### คุณสมบัติหลัก

✅ **VRP Optimization** - คำนวณเส้นทางที่ดีที่สุดอัตโนมัติ  
✅ **Drag & Drop Editor** - แก้ไขแผนด้วยการลากวาง  
✅ **Real-time Map** - ดูแผนที่แบบ real-time  
✅ **Multi-warehouse** - รองรับหลายคลังสินค้า  
✅ **Batch Operations** - จัดการหลายแผนพร้อมกัน

## การสร้างแผนเส้นทางใหม่

### ขั้นตอนที่ 1: เลือกออเดอร์

1. คลิกปุ่ม **"+ สร้างแผนใหม่"** ที่มุมขวาบน
2. เลือก **คลังสินค้า** และ **วันที่จัดส่ง**
3. ระบบจะแสดงรายการออเดอร์ที่พร้อมจัดส่ง
4. เลือกออเดอร์ที่ต้องการ (สามารถเลือกหลายรายการ)
5. คลิก **"ถัดไป"**

**Tips:**
- ใช้ช่องค้นหาเพื่อหาออเดอร์เร็วขึ้น
- ดูรายละเอียดออเดอร์โดยคลิกที่แถว
- เลือกทั้งหมดด้วยปุ่ม "เลือกทั้งหมด"

### ขั้นตอนที่ 2: ตั้งค่า VRP

1. กำหนด **จำนวนจุดส่งสูงสุดต่อเที่ยว** (แนะนำ 8-12 จุด)
2. กำหนด **น้ำหนักสูงสุดต่อเที่ยว** (kg)
3. กำหนด **ปริมาตรสูงสุดต่อเที่ยว** (m³)
4. เลือก **เวลาเริ่มต้น** และ **เวลาสิ้นสุด**
5. เลือก **อัลกอริทึม** (แนะนำ Genetic Algorithm)
6. คลิก **"คำนวณเส้นทาง"**

**การตั้งค่าแนะนำ:**
```
จุดส่งสูงสุด: 10 จุด
น้ำหนักสูงสุด: 1,000 kg
ปริมาตรสูงสุด: 10 m³
เวลา: 08:00 - 18:00
อัลกอริทึม: Genetic Algorithm
```

### ขั้นตอนที่ 3: ดูผลลัพธ์

1. รอระบบคำนวณ (ประมาณ 30 วินาที - 5 นาที)
2. ดูผลลัพธ์ที่แสดง:
   - จำนวนเที่ยว
   - ระยะทางรวม
   - เวลาโดยประมาณ
   - แผนที่เส้นทาง
3. ตรวจสอบความถูกต้อง
4. คลิก **"บันทึกแผน"** เพื่อยืนยัน

**ตัวอย่างผลลัพธ์:**
```
✅ สร้างแผนสำเร็จ!

📊 สรุป:
- จำนวนเที่ยว: 5 เที่ยว
- จำนวนจุดส่ง: 47 จุด
- ระยะทางรวม: 245 km
- เวลาโดยประมาณ: 8 ชั่วโมง
- ประหยัดระยะทาง: 35% (เทียบกับไม่ optimize)
```

## การแก้ไขแผนเส้นทาง

### เปิด Editor

1. คลิกปุ่ม **"แก้ไข"** (✏️) ที่แถวของแผนที่ต้องการ
2. รอระบบโหลดข้อมูล
3. Editor จะเปิดขึ้นมาแบบ full-screen

### การใช้งาน Excel Editor

Editor มีลักษณะคล้าย Excel ที่คุณสามารถ:

#### 1. ย้ายจุดส่งระหว่างเที่ยว
```
วิธีที่ 1: Drag & Drop
- คลิกค้างที่แถวจุดส่ง
- ลากไปวางที่เที่ยวใหม่

วิธีที่ 2: Cut & Paste
- เลือกจุดส่ง
- กด Ctrl+X (Cut)
- คลิกที่เที่ยวปลายทาง
- กด Ctrl+V (Paste)
```

#### 2. เปลี่ยนลำดับจุดส่ง
```
- คลิกค้างที่แถวจุดส่ง
- ลากขึ้น/ลงเพื่อเปลี่ยนลำดับ
- ระบบจะ re-number ลำดับอัตโนมัติ
```

#### 3. เพิ่มจุดส่งจาก Draft
```
- คลิกปุ่ม "เพิ่มจาก Draft"
- เลือกออเดอร์ที่ต้องการ
- เลือกเที่ยวปลายทาง
- คลิก "เพิ่ม"
```

#### 4. แบ่งจุดส่ง (Split Stop)
```
- คลิกขวาที่จุดส่ง
- เลือก "แบ่งจุดส่ง"
- ระบุจำนวนที่ต้องการแบ่ง
- เลือกเที่ยวปลายทาง
- คลิก "ยืนยัน"
```

#### 5. ย้ายข้ามแผน (Cross-plan Transfer)
```
- คลิกขวาที่จุดส่ง
- เลือก "ย้ายไปแผนอื่น"
- เลือกแผนปลายทาง
- คลิก "ยืนยัน"
```

### บันทึกการเปลี่ยนแปลง

1. คลิกปุ่ม **"บันทึก"** (💾) ที่มุมขวาบน
2. รอระบบบันทึก (ประมาณ 2-5 วินาที)
3. ระบบจะแสดงข้อความ "บันทึกสำเร็จ"
4. คลิก **"ปิด"** เพื่อกลับหน้าหลัก

**Keyboard Shortcuts:**
- `Ctrl+S` - บันทึก
- `Ctrl+Z` - Undo
- `Ctrl+Y` - Redo
- `Esc` - ปิด Editor
- `Ctrl+F` - ค้นหา

## การลบแผนเส้นทาง

### ตรวจสอบก่อนลบ

ก่อนลบแผน ระบบจะตรวจสอบว่า:
- ✅ แผนยังไม่ถูกใช้งาน (สถานะ Draft)
- ✅ ไม่มี picklist ที่เชื่อมโยง
- ✅ ไม่มี loadlist ที่เชื่อมโยง

### ขั้นตอนการลบ

1. คลิกปุ่ม **"ลบ"** (🗑️) ที่แถวของแผน
2. ระบบจะแสดง dialog ยืนยัน
3. อ่านข้อความเตือนให้ละเอียด
4. พิมพ์ **"ยืนยัน"** ในช่อง (ถ้าจำเป็น)
5. คลิก **"ลบ"** เพื่อยืนยัน

**⚠️ คำเตือน:**
- การลบแผนไม่สามารถกู้คืนได้
- ออเดอร์จะกลับไปสถานะ Draft
- ข้อมูล metrics จะถูกลบด้วย

## การดูแผนที่

### เปิดแผนที่

1. คลิกปุ่ม **"แผนที่"** (🗺️) ที่แถวของแผน
2. แผนที่จะเปิดขึ้นมาแบบ modal

### ฟีเจอร์แผนที่

#### 1. ดูเส้นทาง
- เส้นทางแต่ละเที่ยวจะแสดงด้วยสีต่างกัน
- คลิกที่เส้นทางเพื่อดูรายละเอียด

#### 2. ดูจุดส่ง
- จุดส่งแสดงเป็น marker บนแผนที่
- คลิกที่ marker เพื่อดูข้อมูลลูกค้า

#### 3. Zoom & Pan
- ใช้ mouse wheel เพื่อ zoom in/out
- คลิกค้างเพื่อเลื่อนแผนที่

#### 4. Filter เที่ยว
- เลือกเที่ยวที่ต้องการดูจาก dropdown
- แผนที่จะแสดงเฉพาะเที่ยวที่เลือก

## การส่งออกข้อมูล

### Export เป็น Excel

1. เลือกแผนที่ต้องการ export (checkbox)
2. คลิกปุ่ม **"Export"** ที่ด้านบน
3. เลือกรูปแบบ:
   - **สรุปแผน** - ข้อมูลภาพรวม
   - **รายละเอียดเที่ยว** - ข้อมูลแต่ละเที่ยว
   - **รายละเอียดจุดส่ง** - ข้อมูลแต่ละจุดส่ง
   - **ทั้งหมด** - ข้อมูลครบถ้วน
4. คลิก **"Download"**

### ไฟล์ Excel จะมี Sheets:

1. **Summary** - สรุปภาพรวม
2. **Trips** - รายการเที่ยว
3. **Stops** - รายการจุดส่ง
4. **Items** - รายการสินค้า
5. **Metrics** - ตัวเลขสถิติ

## คำถามที่พบบ่อย (FAQ)

### Q1: ทำไมระบบคำนวณเส้นทางช้า?
**A:** การคำนวณใช้เวลาขึ้นอยู่กับ:
- จำนวนออเดอร์ (ยิ่งเยอะยิ่งช้า)
- อัลกอริทึมที่เลือก (Genetic ช้าแต่แม่นกว่า Greedy)
- ปกติใช้เวลา 30 วินาที - 5 นาที

### Q2: สามารถแก้ไขแผนที่ Active แล้วได้ไหม?
**A:** ได้ แต่ต้องระวัง:
- ถ้ามี picklist แล้ว จะต้องยกเลิก picklist ก่อน
- ถ้ามี loadlist แล้ว จะแก้ไขไม่ได้

### Q3: ทำไมบางออเดอร์ไม่แสดงในรายการ?
**A:** เพราะ:
- ออเดอร์ไม่ใช่สถานะ Confirmed
- ออเดอร์อยู่ในแผนอื่นแล้ว
- วันที่จัดส่งไม่ตรงกับที่เลือก

### Q4: จะ optimize แผนใหม่ได้ไหม?
**A:** ได้ โดย:
- เปิด Editor
- คลิกปุ่ม "Re-optimize"
- ระบบจะคำนวณใหม่ตาม settings เดิม

### Q5: ข้อมูลจะหายไหมถ้าปิดเบราว์เซอร์?
**A:** ไม่หาย เพราะ:
- ข้อมูลบันทึกบน server
- แต่การแก้ไขที่ยังไม่ Save จะหาย

## การแก้ปัญหา

### ปัญหา: ไม่สามารถสร้างแผนได้

**อาการ:** คลิก "คำนวณเส้นทาง" แล้วไม่มีอะไรเกิดขึ้น

**วิธีแก้:**
1. ตรวจสอบว่าเลือกออเดอร์แล้ว (อย่างน้อย 1 รายการ)
2. ตรวจสอบ settings ว่าถูกต้อง
3. ลอง refresh หน้าเว็บ (F5)
4. ลอง logout แล้ว login ใหม่

### ปัญหา: Editor ช้า/ค้าง

**อาการ:** Editor ใช้งานช้า หรือค้างบ่อย

**วิธีแก้:**
1. ปิด tab อื่นๆ ที่ไม่ใช้
2. Clear browser cache
3. ใช้เบราว์เซอร์ที่แนะนำ (Chrome, Edge)
4. ตรวจสอบ internet connection

### ปัญหา: แผนที่ไม่แสดง

**อาการ:** คลิกดูแผนที่แล้วไม่มีอะไรแสดง

**วิธีแก้:**
1. ตรวจสอบว่าออเดอร์มีพิกัด (latitude, longitude)
2. ลอง refresh หน้าเว็บ
3. ตรวจสอบ browser console (F12) ว่ามี error

### ปัญหา: บันทึกไม่ได้

**อาการ:** คลิก "บันทึก" แล้วไม่มีอะไรเกิดขึ้น

**วิธีแก้:**
1. ตรวจสอบว่ามีการเปลี่ยนแปลงจริง
2. ตรวจสอบว่าไม่มี validation error
3. ลอง Save อีกครั้ง
4. ถ้ายังไม่ได้ ลอง refresh แล้วแก้ไขใหม่
