### 5.1 สิ่งที่ไม่ต้องแก้ไข

### 5.2 สิ่งที่ควรแก้ไข

#### ลำดับความสำคัญ 1: สต็อกศูนย์มีการจอง
```sql
-- ปล่อยการจองที่ไม่สามารถ fulfill ได้
UPDATE wms_inventory_balances
SET reserved_pack_qty = 0, reserved_piece_qty = 0
WHERE balance_id IN (25285, 28359)
  AND total_piece_qty = 0;
```

#### ลำดับความสำคัญ 2: พาเลทหลาย Location
- ตรวจสอบพาเลท ATG... ว่าเป็นการย้ายบางส่วนหรือข้อมูลผิดพลาด
- หากเป็นข้อมูลผิดพลาด ให้รวมสต็อกไปที่ Location ที่ถูกต้อง

#### ลำดับความสำคัญ 3: Ledger vs Balance
- ไม่แนะนำให้แก้ไขทันที เนื่องจาก Balance > Ledger (มีสต็อกมากกว่า)
- ควรทำ Physical Stock Count เพื่อยืนยันยอดจริง
- หากยอดจริงตรงกับ Balance ให้สร้าง Ledger entry เพื่อ reconcile
