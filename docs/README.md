# Documentation Structure

โครงสร้างเอกสารของโปรเจกต์ AustamGood WMS

## 📁 โครงสร้างโฟลเดอร์

### `/docs/analysis/`
เอกสารวิเคราะห์และการวางแผนต่างๆ
- `FREIGHT_RATE_IMPROVEMENTS.md` - การปรับปรุงระบบคำนวณค่าขนส่ง
- `LOADLIST_PICKLIST_IMPROVEMENTS.md` - การปรับปรุง Loadlist และ Picklist
- `PICKLIST_STOCK_TRANSFER_ANALYSIS.md` - การวิเคราะห์การโอนสต็อกผ่าน Picklist
- `TRIGGER_API_ANALYSIS.md` - การวิเคราะห์ Trigger API
- `TRIGGER_MIGRATION_PLAN.md` - แผนการ Migrate Triggers

### `/docs/fixes/`
เอกสารสรุปการแก้ไขปัญหาต่างๆ
- `BALANCE_SYNC_FIX_SUMMARY.md` - สรุปการแก้ไขปัญหา Balance Sync
- `FIXES_IMPLEMENTATION_SUMMARY.md` - สรุปการแก้ไขทั้งหมด
- `WORKFLOW_FIX_SUMMARY.md` - สรุปการแก้ไข Workflow
- `WORKFLOW_FIXES_COMPLETED.md` - รายการแก้ไข Workflow ที่เสร็จแล้ว

### `/docs/guides/`
คู่มือการใช้งานและพัฒนา
- `DEPLOYMENT_GUIDE.md` - คู่มือการ Deploy
- `DESIGN_SYSTEM.md` - ระบบออกแบบ UI/UX

### `/docs/reports/`
รายงานการตรวจสอบระบบ
- `SYSTEM_AUDIT_REPORT_2025-11-29.md` - รายงานตรวจสอบระบบ
- `WORKFLOW_AUDIT_REPORT.md` - รายงานตรวจสอบ Workflow

### `/docs/` (Root Level)
เอกสารหลักและเอกสารทั่วไป
- `DATABASE_SETUP.md` - การตั้งค่าฐานข้อมูล
- `DOCUMENTATION_INDEX.md` - ดัชนีเอกสารทั้งหมด
- `INVENTORY_BALANCE_SYNC.md` - ระบบ Sync ยอดสินค้า
- `ORDER_IMPORT_ANALYSIS.md` - การวิเคราะห์การนำเข้าออเดอร์
- `ORDER_IMPORT_SUMMARY.md` - สรุปการนำเข้าออเดอร์
- `SHIPPING_COST_CALCULATION.md` - การคำนวณค่าขนส่ง
- `VRP_*.md` - เอกสารเกี่ยวกับระบบ Vehicle Routing Problem

## 📄 ไฟล์สำคัญใน Root
- `README.md` - เอกสารหลักของโปรเจกต์
- `CLAUDE.md` - คำแนะนำสำหรับ Claude AI
- `GEMINI.md` - คำแนะนำสำหรับ Gemini AI
