# Audio Files for Mobile App

This directory contains audio files for user feedback in mobile app.

## Required Files:

- `success.mp3` - Sound for successful scan
- `error.mp3` - Sound for scan error
- `beep.mp3` - Sound for button press

## How to add:

Place your MP3 files in this directory. The app will automatically use them.

## Recommendations:

- Keep files small (<100KB)
- Use short sounds (0.5-1 second)
- Clear, distinct sounds
- Medium volume

## Free sound sources:

- https://freesound.org/
- https://mixkit.co/free-sound-effects/
